Bootstrap

Notes: Resourceful web that provides a cheatsheet on how to do basic animations and/or interactions in bootstrap that interact with your html file. Comands are more natural language related than maybe CSS for example.

Tailwind

Notes: Tailwind is a utility-first CSS framework for rapidly building custom user interfaces. Although it is not a traditional CSS framework like Bootstrap, it provides a set of pre-defined classes that can be used to style HTML elements directly in the markup. This approach allows for greater flexibility and customization, as developers can combine multiple utility classes to achieve the desired design without writing custom CSS. Considered a bit pricy.
