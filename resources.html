Bootstrap

Notes: Resourceful web that provides a cheatsheet on how to do basic animations and/or interactions in bootstrap that interact with your html file. Comands are more natural language related than maybe CSS for example.

Tailwind

Notes: Tailwind is a utility-first CSS framework for rapidly building custom user interfaces. Although it is not a traditional CSS framework like Bootstrap, it provides a set of pre-defined classes that can be used to style HTML elements directly in the markup. This approach allows for greater flexibility and customization, as developers can combine multiple utility classes to achieve the desired design without writing custom CSS. Considered a bit pricy.

Jekyll

Notes: Jekyll is a static site generator that allows developers to create websites using simple templates and markdown files. It is often used for blogs and documentation sites, as it provides a straightforward way to manage content without the need for a database. Jekyll is built on Ruby and integrates well with GitHub Pages, making it easy to deploy and host static websites directly from a GitHub repository. Websites have little to no animations, depending on which one.

GSAP

Notes: GSAP (GreenSock Animation Platform) is a powerful JavaScript library for creating high-performance animations on the web. It provides a robust set of tools and features that allow developers to create complex animations with ease. GSAP is known for its speed and efficiency, making it ideal for creating smooth, interactive animations that work across all major browsers. It supports a wide range of animation types, including tweens, timelines, and scroll-based animations. GSAP also offers advanced features like physics-based animations and SVG morphing, making it a versatile choice for web developers looking to enhance the user experience with engaging animations.

HTML5up

Notes: Html45up is a collection of free, fully responsive HTML5 and CSS3 site templates and themes. These templates are designed to be modern, clean, and easy to customize, making them ideal for a wide range of web projects. HTML5up templates often include features such as flexible grid layouts, mobile-friendly designs, and various UI components that can be easily adapted to fit the needs of different websites. They are a great resource for developers looking to quickly create professional-looking websites without starting from scratch.
